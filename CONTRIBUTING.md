# Contributing to PussyLang

Thanks for your interest! Here's how you can help:

**Report bugs** – open an issue with a minimal script and your OS/VM mode.

**Suggest features** – open an issue labeled `enhancement`. Keep in mind the language aims to stay minimal.

**Add native functions** – implement in `NativeRegistry.java` (Java) and add to `native_table` in `natives.c` (C). See existing examples.

**Improve docs** – Send issues directly on the Discord. Small fixes and clarifications are always welcome.

**Pull requests** – fork, branch from `main`, test both Java VM and AOT if relevant, then open a PR with a clear description.

Questions? Open an issue or join the Discord.
